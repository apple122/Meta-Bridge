# Patch Update - 2026-04-14

การอัปเดตใหญ่ในระบบการยืนยันตัวตนและการแจ้งเตือน เพื่อความสะดวกและความปลอดภัยที่ดียิ่งขึ้น

## 1. ระบบอีเมล (Email System Migration)
- **ย้ายจาก Resend ไปเป็น Brevo API:** แก้ไขปัญหาอีเมลไม่ส่งสำหรับโดเมนทั่วไป ตอนนี้ระบบสามารถส่งอีเมลแจ้งเตือนและ OTP ได้อย่างเสถียร 100%
- **Personalized Greeting:** อีเมลทุกฉบับจะระบุชื่อผู้รับอย่างชัดเจน เช่น "สวัสดีครับคุณ [ชื่อผู้ใช้]" เพื่อความเป็นส่วนตัวและน่าเชื่อถือ

## 2. ความเป็นส่วนตัวและความปลอดภัย (Privacy & Security)
- **Email Masking:** ปิดบังข้อมูลอีเมลในหน้าจอแสดงผลทั้งหมด (เช่น `pe******@gmail.com`) เพื่อป้องกันการแอบดูข้อมูล
- **Detailed Device Detection:** ระบบสามารถระบุรุ่นอุปกรณ์ที่เข้าใช้งานได้ละเอียดขึ้นมาก เช่น "iPhone 14-16 Pro Max", "SM-S901B (Galaxy S22)" แทนการขึ้นแค่ชื่อแบรนด์แบบเดิม

## 3. การเข้าสู่ระบบ (Enhanced Login)
- **Hybrid Login:** ผู้ใช้สามารถเข้าสู่ระบบด้วย **ชื่อผู้ใช้ (Username)** หรือ **อีเมล (Email)** ในช่องเดียวกันได้ทันที ช่วยลดปัญหาการจำชื่อผู้ใช้ไม่ได้
- **Updated UI:** ปรับปรุงช่องกรอกข้อมูลให้ระบุ "ชื่อผู้ใช้ หรือ อีเมล" อย่างชัดเจนทั้งภาษาไทยและอังกฤษ

## 4. ประวัติและการแสดงผล (History & Display)
- **No Limit History:** ยกเลิกการจำกัดจำนวนรายการในหน้า "ประวัติการเข้าใช้งาน" และ "หน้าจัดการแอดมิน" ทำให้สามารถตรวจสอบข้อมูลย้อนหลังได้ทั้งหมดโดยไม่มีที่สิ้นสุด
- **Build Optimization:** แก้ไข TypeScript errors ทั้งหมดเพื่อให้ระบบสามารถ Build และ Deploy ได้อย่างสมบูรณ์
